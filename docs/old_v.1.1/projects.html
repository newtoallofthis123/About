NoobNote

NoobNote is a simple light weight alternative to notepad. No telementry and private. It is very feature rich. NoobNote is written purely in python and written using the tkinter framework. Everything is done locally, it is light weight, comes with a calendar and a qr-code generator and many other tools.

You can get NoobNote from my github. It comes in 3 forms: Setup.exe, NoobNote.zip and NoobNote.py. Setup.exe is recommended for beginners. Although, if you want to get involved or want to see how it works, it highly recommended you download the source code. NoobNote can be run from the source code with python NoobNote.py. Hope you enjoy using it.

Leaf

Leaf is a simple Qr-Code Generator written in python. It is simple and efficient. You can easily generate QR-Codes, delete them and directly open them from the application

Leaf is also written in python and tkinter and utillizes a trusted third party moducle called py-qrcode. If you decide to install leaf, you can try the zip or the setup.exe, no need to have python to run them though. If you decide to install and try from the source code, you do need python and py-qrcode install (pip install pyqrcode)
