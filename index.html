    Some of My Projects

    NoobNote

    NoobNote is a simple Notepad written in python.
    It aims to add new features to Notepad while maintianing the Notepad nostalgia.
    NoobNote is registered under the MIT License. NoobNote is simple and beginner friendly.
    Hope you like it. You can see it here.

    pip install NoobNote

    ToPy

    ToPy is a CLI ToDo manager.
    It is very easy and simple to use. ToPy uses sqlite3 to store all your ToDo's.
    ToPy is registered under the MIT License. You can add Topy to your path and easily use it.
    If you are an avid user of the terminal interface, you will like ToPy.

    Unleaky

    Unleaky is a CLI Password manager.
    Unleaky uses sqlite3 to store all your Passowrds, hence it is not recommended for a few.
    ToPy is registered under the MIT License.
    Unleaky comes with a deflaut master passowrd that can be changed. More info on the website.
    Unleaky also comes with a master password that you can change.
    Unleaky is my attempt to use and learn sqlite3 databases.

    leaf Qr

    Leaf is a simple QrCode Generator and Manager
    Leaf uses PyPng and PyQr to generate and manage your QrCodes with a tkinter window.
    Leaf is registered under the MIT License.
    Leaf also can be conventienly launched from the terminal if installed with pip
    If you are interested in this beginner friendly project, defiently give it a try.

    pip install leafQR
